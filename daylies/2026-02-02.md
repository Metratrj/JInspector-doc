# Projektziel

**JByteInspector** ist ein produktionsreifer **Java Opcode & Bytecode Analyzer** mit klarer Modultrennung,
reproduzierbarem Build, Tests, Benchmarks und CI/CD.

### Technologische Vorgaben

- Java **25**
- Build-System: Gradle **9.2.0**
- Repository: GitHub
- Versioning: Git + Release (SemVer)
- Testing: JUnit 5
- Benchmarking: JMH
- Local Storage: SQLite
- CI/CD: GitHub Actions
- Multi-Module Monorepo
